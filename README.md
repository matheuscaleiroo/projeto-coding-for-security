1. Introdução

Neste checkpoint, o aluno aplicará os conhecimentos de Python e segurança para criar uma
ferramenta CLI que automatize uma tarefa real de TI ou cibersegurança.
Para o CP 3, a entrega será prioritariamente em duplas. Caso não seja possível poderá ser feita
individualmente.
A avaliação consistirá em um script Python de linha de comando (CLI), cujo propósito será criar uma
ferramenta que automatize uma tarefa relacionada a um dos temas estudados nas disciplinas do
semestre. O tema e a ferramenta devem estar dentro do universo da tecnologia.
